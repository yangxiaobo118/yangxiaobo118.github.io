git命令

常用git命令

首次使用

  • git init 初始化
  • git add . 添加文件到git
  • git commit -m “first commit” 提交更新
  • git remote add origin https://github.com/yangxiaobo118/SpringCloundDemo.git 链接远端
  • git push -u origin master 将本地的仓库的文件提交到github

    修改更新时，记得要先pull

  • git status 查看状态
  • git pull origin master 拉取更新
  • git add -a -m “comment” 添加
  • git commit -a 提交
  • git push -u origin master 推送到远程服务器
